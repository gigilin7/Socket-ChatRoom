## Socket
+ 又稱為「插座」
+ 網路上互相溝通連結的端點
+ 透過網路進行通訊
+ 提供介面讓使用者不需處理下層的網路協定(傳輸層、網路層)，只需專注開發(應用層)程式設計

## 使用Client/Server架構
+ 建立一個server端與多個client端進行通訊連接
+ server端用java圖形介面呈現
 
## 注意
+ ip位址會隨著server端建立的位置而改變
+ 連接埠(port)的範圍從0到65535，可自選，不一定要與此相同
